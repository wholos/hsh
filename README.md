#

How to install!?
``` bash
sudo pacman -S git make ghc ghc-libs ghc-filesystem ghc-static
git clone https://github.com/wholos/hsh/
cd hsh/
ghc-pkg recache
make all
```

How tu run!?
```./main``` or ```sudo cp main /usr/bin/hsh``` after ```hsh```
